# 物归 — 家庭物品清单管理小程序

基于 PRD 开发的微信小程序，帮助家庭快速建立物品数字化清单，支持空间管理、过期提醒和家庭共享。

## 快速开始

1. 打开**微信开发者工具**
2. 选择「导入项目」，项目目录指向本文件夹
3. AppID 填写你的小程序 AppID（或使用测试号）
4. 点击确定即可预览

> 首次打开会自动初始化示例数据（3 个空间、7 个物品），方便快速体验功能。可在「我的 → 清空数据」重置。

## 技术说明

- **数据存储**：使用 `wx.setStorageSync` 实现本地数据层（`utils/store.js`），接口设计为可无缝替换为云端 API
- **设计系统**：陶土橙（#C0623E）为主色，暖米色背景，温暖有序的家居工具风格
- **性能**：CSS 变量驱动设计 token，`lazyCodeLoading` 按需加载组件，setData 精简

## 后续接入真实后端

将 `utils/store.js` 中的本地存储函数替换为 `wx.request` 调用即可，接口签名保持不变：

```javascript
// 当前（本地）
getItems() { return read(STORAGE_KEYS.ITEMS, []); }

// 替换为（云端）
async getItems() {
  const res = await wx.request({ url: BASE_URL + '/items', ... });
  return res.data;
}
```
